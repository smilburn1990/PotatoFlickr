#Flickr Public Feed

Flickr public gallery with search functionality 

#Requirements

Node.js versions >= 0.8.0

#Instructions

-clone repository<br>
-$cd potatoFlickr <br>
-$npm install <br>
-$grunt serve

#External resources

Flickr public API - https://api.flickr.com/services/feeds/photos_public.gne

Run grunt for building and '$grunt serve' for preview.
